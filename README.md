## About me

I'm a software developer focused on **full-stack web development** — frontend interfaces that users enjoy, and backend systems that hold up under real conditions.

I founded **Swiftmere** to bring modern, scalable digital products to startups and growing businesses in Latin America and beyond. I work across the entire stack: Vue 3 SPAs, .NET Web APIs, Clean Architecture, and production-ready deployment pipelines.

What drives me isn't just writing code — it's understanding *why* an architecture decision matters, *how* a UI choice affects conversion, and *when* the right solution is actually the simpler one.

## How I Work

I care about **architecture that makes sense**, not just architecture that sounds impressive. Before writing a line of code I try to understand the domain, define the contracts, and think about what will need to change six months from now.

On the frontend, I think like a designer: whitespace is intentional, interactions have purpose, and performance is not optional.
